Installing Invenio

Installing Invenio is a simple task, you only need to use pip from your command line and...a few configuration settings:

$ pip install invenio


If you already have the source code, you can easily start a demo with Docker:

$ docker-compose build
$ docker-compose up


You can find both source code and Python packages on GitHub and pypi.

Developing for Invenio

Invenio 3 is a multi-modular framework. You can find all of Invenio's modules on GitHub. We do keep a "Build status" list on our GitHub Wiki as well!


We encourage you to have a look at the code and if you feel like hacking, go ahead and contribute. Feel free to clone our repo:

$ git clone git@github.com:inveniosoftware/invenio.git
