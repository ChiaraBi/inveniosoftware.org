Developing for Invenio

Invenio 3 is a multi-modular framework. You can find all of Invenio's modules on GitHub. We do keep a "Build status" list on our GitHub Wiki as well! You should know that we explit our modules in three organistations:

  • inveniosoftware: yada yada
  • inveniosoftware-contrib: yada yada
  • inveniosoftware-attic: yada yada

We encourage you to have a look at the code and if you feel like hacking, go ahead and contribute. Feel free to clone any of our modules:

$ git clone git@github.com:inveniosoftware[-contrib|-attic]/module_name.git
